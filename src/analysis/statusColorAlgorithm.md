## Анализ библиотеки color-convert

### Основные возможности:
1. **Конвертация между цветовыми пространствами:**
   - RGB ↔ HSL ↔ HSV ↔ HWB
   - CMYK, XYZ, LAB, LCH
   - HEX, Keywords, ANSI

2. **Алгоритм маршрутизации:**
   - Поиск кратчайшего пути между цветовыми пространствами
   - Автоматическое построение цепочки конвертаций

### Принципы работы:
1. **Граф конвертаций** - каждая модель цвета связана с другими
2. **BFS алгоритм** - поиск кратчайшего пути между моделями
3. **Цепочка конвертаций** - автоматическое построение промежуточных шагов

## Предполагаемый алгоритм подбора статусных цветов

### Основные принципы:
1. **От primary цвета к статусным:**
   - Success (зеленый) - смещение hue на +120°
   - Warning (желтый) - смещение hue на +60°
   - Error (красный) - смещение hue на +180° или -180°

2. **Сохранение контраста:**
   - Адаптация lightness для сохранения читаемости
   - Корректировка saturation для гармонии

3. **Доступность:**
   - Проверка контраста с фоном
   - Автоматическая корректировка при недостаточном контрасте
