# Fast UI

Современный веб-проект с быстрым и отзывчивым пользовательским интерфейсом.

## Описание

Fast UI - это проект, направленный на создание быстрых, современных и удобных веб-интерфейсов.

## Установка и запуск

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build
```

## Структура проекта

```
Fast UI/
├── src/           # Исходный код
├── public/        # Статические файлы
├── docs/          # Документация

```
